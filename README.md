🐍 Snake Neon Gamer

Um jogo clássico da cobrinha reinventado com estilo Neon Cyberpunk, desenvolvido em JavaScript + HTML5 Canvas.




🚀 Demonstração

👉 Em breve (GitHub Pages)

🎮 Funcionalidades

🐍 Movimento fluido da cobra

🍎 Sistema de pontuação

💥 Partículas ao coletar itens

🌌 Visual Neon Gamer (Glow + Grid)

⚡ Efeito de rastro (fade)

🎯 Colisão com o próprio corpo

🔁 Reinício automático ao perder

🧠 Tecnologias utilizadas
JavaScript (Vanilla)
HTML5 Canvas
CSS3 (estilização neon)

🕹️ Como jogar
⬅️ ➡️ ⬆️ ⬇️ → mover a cobra
Comer a maçã para crescer
Evitar colidir com o próprio corpo

💻 Como rodar o projeto
Clone o repositório:
git clone https://github.com/seu-usuario/snake-neon-game.git
Acesse a pasta:
cd snake-neon-game
Abra o arquivo:
index.html
